# PreviewExample

## 示意流程

* FPreviewScene : 用于维护预览世界里的对象，光照等等。
* FEditorViewportClient : 用于控制FPreviewScene中的逻辑,如Actor的位置等。
* SViewportToolBar : 预览窗口上的工具栏，如缩放、相机移动速度等等。
* SEditorViewport : 用于维护布局，并控制FEditorViewportClient和SViewportToolBar的创建。
